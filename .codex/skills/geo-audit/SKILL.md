# geo-audit

Use this skill when the user wants a complete single-page GEO + SEO audit.

## Workflow
1. Execute `python run.py <url> --format json`.
2. Summarize scores for `seo`, `geo`, and `technical`.
3. Group the response into strengths, weaknesses, and highest-priority fixes.
4. Quote recommendations from the generated check data rather than inventing a second scoring system.

## Deliverable structure
- Executive summary
- Category scores
- Top priorities
- Recommended next actions
